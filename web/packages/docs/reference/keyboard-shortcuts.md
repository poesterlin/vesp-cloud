# Keyboard Shortcuts

Keyboard shortcuts available in the VESP Cloud editor. Shortcuts are disabled
when focus is inside a form input (`<input>`, `<textarea>`, or `contentEditable`
elements).

## Canvas editing

| Shortcut | Action |
|----------|--------|
| **Delete** / **Backspace** | Delete selected component(s) |
| **Arrow keys** | Nudge selected component(s) by 1&nbsp;px |
| **Shift** + **Arrow keys** | Nudge selected component(s) by 10&nbsp;px |
| **Alt** + drag | Duplicate the selected component(s) and drag the duplicates |
| **Ctrl** + **C** | Copy selected component(s) to clipboard |
| **Ctrl** + **V** | Paste copied component(s) with a 12&nbsp;px offset |
| **Ctrl** + **Z** | Undo |
| **Ctrl** + **Shift** + **Z** | Redo |
| **Ctrl** + **Y** | Redo (alternative) |

## Modals and dialogs

| Shortcut | Action |
|----------|--------|
| **Escape** | Close any open modal or dialog |

## Template input

When editing a text widget's template content:

| Shortcut | Action |
|----------|--------|
| Type **two opening curly braces** in a text field | Open the entity picker modal |
| **ArrowLeft** at start of text | Move focus to the previous binding chip |
| **ArrowRight** at end of text | Move focus to the next binding chip |
| **Backspace** at start of text field | Remove the preceding binding chip |
| **Delete** at end of text field | Remove the following binding chip |
| **Backspace** / **Delete** on a chip | Remove the chip |

## Icon searcher

| Shortcut | Action |
|----------|--------|
| **ArrowUp** / **ArrowDown** | Navigate between icons |
| **Enter** | Select the highlighted icon |
| **Escape** | Close the icon searcher |

## Service picker

| Shortcut | Action |
|----------|--------|
| **Enter** | Submit a custom service name |
| **Escape** | Close the service picker |

## Page list

| Shortcut | Action |
|----------|--------|
| **Enter** | Save a renamed page name |
| **Escape** | Cancel renaming |

## Undo and redo

The editor supports up to 50 undo steps. Each undo step captures a full snapshot
of the project state, including which dashboard page or detail view was open.

The toolbar also has **Undo** and **Redo** buttons with tooltips showing the
keyboard shortcuts.
